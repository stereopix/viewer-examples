This example shows a simple case with only a video.

The short video clip was created starting with a wonderful work by Daniel Bystedt published under CC-BY-SA 3.0 license. The original blender project can be found on the website of blender as demo file under the name "Blender 2.90 - Splash Fox". The derived blender file was accidently deleted, but it only added camera movement and enabled stereoscopic views with adjusted convergence plane and interocular distance.

In this example, the video file is prepared in two versions, one as a WebM container with a VP9 stream, and another as a MP4 version with AVC stream added as fallback. Neither file has audio track. Two versions are not necessary.
